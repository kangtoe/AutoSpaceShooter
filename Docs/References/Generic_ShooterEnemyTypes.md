# 일반적인 슈팅 게임 적 타입 분석

## 개요
슈팅 게임에서 자주 사용되는 적 타입들을 역할별로 분류하고 분석한 참고 문서입니다. 각 적의 역할, 행동 패턴, 위협도, 카운터 전략, 디자인 포인트를 정리하여 우리 프로젝트의 적 설계에 참고할 수 있도록 합니다.

**참고 노트**: 이 문서는 Nova Drift를 포함한 다양한 슈팅 게임의 적 타입을 일반화한 것입니다. 실제 Nova Drift의 26종 적 분석은 [NovaDrift_EnemyTypes.md](NovaDrift_EnemyTypes.md)를 참조하세요.

**적 타입 구성 예시**:
- 기본 적: 8종 (소형 2종, 중형 3종, 대형 3종)
- 특수 적: 엘리트 변종 (모든 타입의 강화판) + 보스

---

## 1. 적 분류 체계

### 1.1 크기별 분류
- **소형 (Small)** - 2종: 체력 낮음, 빠른 이동, 주로 그룹 스폰
- **중형 (Medium)** - 3종: 중간 체력, 특수 능력, 다양한 역할
- **대형 (Large)** - 3종: 높은 체력, 강력한 공격, 단독 위협
- **초대형 (Huge)** - 미니보스급: 매우 높은 체력, 복합 패턴

### 1.2 역할별 분류
- **스워머 (Swarmer)**: 물량으로 압박
- **탱커 (Tank)**: 높은 체력으로 시선 끌기
- **딜러 (Dealer)**: 원거리 공격
- **서포터 (Support)**: 소환 및 증원
- **분열형 (Splitter)**: 죽을 때 추가 적 생성
- **장애물형 (Obstacle)**: 공간 차단 및 이동 방해

---

## 2. 소형 적 (Small Enemies) - 총 2종

Nova Drift의 가장 기본이 되는 적들로, 낮은 체력과 빠른 속도가 특징입니다. 주로 그룹으로 스폰되어 물량으로 압박합니다.

### 2.1 Dart (다트)
**기본 정보**
- **역할**: 기본 적, 스워머
- **크기**: 소형
- **체력**: ★☆☆☆☆ (1-2 히트)
- **이동 속도**: ★★★★☆ (빠름)
- **공격력**: ★☆☆☆☆ (낮음)
- **위협도**: ⭐☆☆☆☆

**행동 패턴**
- 플레이어를 향해 직선으로 돌진
- 목표 지점까지 경로 변경 없음
- 단순하지만 수가 많으면 위협적
- 회피하기 쉽지만 지속적인 압박감 제공

**스폰 특성**
- 그룹으로 스폰 (5-10마리)
- 초반 웨이브부터 등장
- Carrier에 의해 소환되기도 함

**카운터 전략**
- 광역 무기로 한 번에 처리
- 빠른 연사력으로 돌진 전에 제거
- 이동으로 회피 가능

**디자인 포인트**
- 가장 기본이 되는 적으로, 플레이어가 움직임을 학습하는 데 사용
- 단독으로는 위협적이지 않지만 다른 적과 조합될 때 압박감 제공
- 초반 웨이브에서 플레이어에게 "성취감"을 주는 역할

---

### 2.2 Splinter (파편)
**기본 정보**
- **역할**: 분열형 적
- **크기**: 소형
- **체력**: ★☆☆☆☆ (낮음)
- **이동 속도**: ★★★☆☆ (보통)
- **공격력**: ★☆☆☆☆ (낮음)
- **위협도**: ⭐⭐☆☆☆

**행동 패턴**
- 불규칙한 움직임으로 접근
- 죽을 때 2-3개의 작은 파편으로 분열
- 분열된 파편도 플레이어에게 피해 발생
- 연쇄적으로 화면을 채울 수 있음

**스폰 특성**
- 그룹으로 스폰 (3-5마리)
- 중반부터 등장
- 특정 대형 적이 죽을 때 생성되기도 함

**카운터 전략**
- 관통 무기로 분열 전에 여러 마리 처리
- 체인 공격으로 연쇄 제거
- 일정 거리 유지하며 분열 파편 회피

**특수 메커니즘**
- 분열 단계: 2-3단계 (크기 감소)
- 최종 단계 파편은 분열하지 않음
- 각 분열마다 이동 속도 증가

**디자인 포인트**
- "연쇄 반응" 메커니즘으로 시각적 만족감 제공
- 무작정 공격하면 더 위험해지는 학습 요소
- 특정 무기(관통, 체인)의 가치 상승
- 화면을 채우는 효과로 압박감 증폭

## 3. 중형 적 (Medium Enemies) - 총 3종

중간 크기의 적들로, 다양한 역할과 특수 능력을 보유합니다. 원거리 공격, 소환, 장애물 등의 메커니즘으로 전략적 깊이를 제공합니다.

### 3.1 Asteroid (소행성)
**기본 정보**
- **역할**: 장애물형 적
- **크기**: 중형
- **체력**: ★★★☆☆ (중간, 5-8 히트)
- **이동 속도**: ★☆☆☆☆ (느림)
- **공격력**: ★★☆☆☆ (중간)
- **위협도**: ⭐⭐☆☆☆

**행동 패턴**
- 느리게 움직이며 공간 차단
- 일정 방향으로 직선 이동 또는 회전
- 크기가 커서 탄환 흡수 역할
- 일부는 죽을 때 더 작은 조각으로 분열

**스폰 특성**
- 단독 또는 소그룹 (2-3개)
- 초반부터 등장
- 화면 경계에서 스폰

**카운터 전략**
- 고화력 무기로 빠르게 파괴
- 관통 무기로 뒤의 적까지 공격
- 회피하며 다른 적 우선 처리

**디자인 포인트**
- "탄환 흡수" 역할로 플레이어의 전략적 판단 요구
- 분열 옵션으로 예측 불가능성 추가
- 느린 속도로 플레이어에게 여유 제공하지만 공간을 차단하여 압박

---

### 3.2 Bomber (폭격기)
**기본 정보**
- **역할**: 원거리 딜러
- **크기**: 중형
- **체력**: ★★☆☆☆ (중간)
- **이동 속도**: ★★★☆☆ (보통)
- **공격력**: ★★★☆☆ (높음)
- **위협도**: ⭐⭐⭐☆☆

**행동 패턴**
- 플레이어 주변을 원형으로 순회
- 일정 거리(사거리 밖)를 유지하며 회피
- 주기적으로 투사체 발사 (2-3개 동시)
- 직접 접근하지 않음

**투사체 특성**
- 직선 또는 플레이어 방향으로 발사
- 중간 속도
- 벽이나 화면 경계에 닿으면 소멸

**스폰 특성**
- 소그룹 (2-4마리)
- 중반 웨이브부터 등장
- 다른 적과 조합하여 스폰

**카운터 전략**
- 호밍 무기로 원거리 처치
- 빠른 기동력으로 접근하여 근거리 처리
- 투사체 패턴 파악 후 회피

**디자인 포인트**
- 플레이어의 "안전 지대"를 제거하는 역할
- 거리 유지 AI로 플레이어의 접근을 어렵게 만듦
- 다른 적과 함께 나올 때 시너지 효과 극대화 (플레이어를 몰아붙임)

---

### 3.3 Carrier (운반선)
**기본 정보**
- **역할**: 소환형 서포터
- **크기**: 중형-대형
- **체력**: ★★★★☆ (높음)
- **이동 속도**: ★★☆☆☆ (느림)
- **공격력**: ☆☆☆☆☆ (없음, 소환만)
- **위협도**: ⭐⭐⭐⭐☆ (높음 - 방치 시 위험)

**행동 패턴**
- 느리게 움직이며 회피 기동
- 주기적으로 Dart를 2-3마리 소환
- 플레이어를 직접 공격하지 않음
- 방치하면 화면이 적으로 가득 참

**소환 메커니즘**
- 소환 주기: 5-8초
- 한 번에 2-3마리 소환
- 최대 동시 소환 수 제한 (8-10마리)

**스폰 특성**
- 단독 또는 소수 (1-2마리)
- 중반 웨이브부터 등장
- 보스 웨이브에서 보스가 소환하기도 함

**카운터 전략**
- **우선 처치 필수** - 가장 먼저 제거해야 함
- 집중 화력으로 빠르게 파괴
- 소환된 Dart는 무시하고 Carrier 집중

**디자인 포인트**
- "위협 우선순위" 학습을 위한 적
- 방치할 경우 기하급수적으로 난이도 상승
- 플레이어에게 전략적 판단 요구 (소환된 적 vs 소환자)
- 소환 수 제한으로 무한 증식 방지 (밸런스)

## 4. 대형 적 (Large Enemies) - 총 3종

높은 체력과 강력한 공격력을 가진 적들입니다. 각자 독특한 메커니즘을 가지고 있으며, 단독으로도 강력한 위협이 됩니다.

### 4.1 Dreadnought (드레드노트)
**기본 정보**
- **역할**: 탱커
- **크기**: 대형
- **체력**: ★★★★★ (매우 높음, 20-30 히트)
- **이동 속도**: ★☆☆☆☆ (느림)
- **공격력**: ★★★★☆ (매우 높음)
- **위협도**: ⭐⭐⭐⭐☆

**행동 패턴**
- 플레이어를 향해 천천히 접근
- 주기적으로 강력한 돌진 공격
- 충격파로 넉백 효과 발생
- 다른 적의 방패 역할

**공격 패턴**
1. **일반 이동**: 천천히 플레이어 추적
2. **돌진**: 빠른 속도로 직선 돌진 (예고 후)
3. **충격파**: 충돌 시 주변에 넉백 효과

**스폰 특성**
- 단독 스폰
- 중후반 웨이브부터 등장
- 다른 적과 함께 스폰되어 시선 분산

**카운터 전략**
- 지속 피해 무기로 서서히 제거
- 기동전으로 돌진 회피
- 다른 적 먼저 처리 후 집중

**디자인 포인트**
- "시선 끌기" 역할로 다른 적의 공격 기회 제공
- 돌진 예고로 플레이어에게 반응 시간 부여 (공정성)
- 높은 체력으로 장기전 유도, 플레이어의 인내심 테스트
- 충격파로 플레이어의 위치 제어 (게임 다이나믹스)

---

### 4.2 Serpent (서펀트)
**기본 정보**
- **역할**: 기동형 대형 적
- **크기**: 대형 (긴 몸체, 5-8 세그먼트)
- **체력**: ★★★★☆ (세그먼트별로 체력)
- **이동 속도**: ★★★☆☆ (보통)
- **공격력**: ★★☆☆☆ (중간)
- **위협도**: ⭐⭐⭐⭐☆

**행동 패턴**
- 뱀처럼 구불구불하게 이동
- 몸체 전체가 충돌 판정
- 길게 늘어져 회피가 어려움
- 머리부터 순차적으로 파괴 가능

**구조적 특성**
- 여러 세그먼트로 구성
- 각 세그먼트에 독립적인 체력
- 머리가 파괴되면 전체 파괴
- 중간 세그먼트 파괴 시 분리 가능

**스폰 특성**
- 단독 스폰
- 후반 웨이브부터 등장
- 화면 경계에서 진입

**카운터 전략**
- 관통 무기로 여러 세그먼트 동시 타격
- 머리 집중 공격으로 빠른 처치
- 위치 선점으로 회피 공간 확보

**디자인 포인트**
- 세그먼트 구조로 시각적 독특함 제공
- "공간 점유" 메커니즘으로 플레이어 이동 제한
- 머리 타겟팅으로 플레이어의 정밀 조준 장려
- 세그먼트별 파괴로 점진적 약화 (만족감)

---

### 4.3 Leviathan (리바이어던)
**기본 정보**
- **역할**: 미니 보스급 적
- **크기**: 초대형
- **체력**: ★★★★★ (극도로 높음, 50+ 히트)
- **이동 속도**: ★☆☆☆☆ (매우 느림)
- **공격력**: ★★★★★ (극도로 높음)
- **위협도**: ⭐⭐⭐⭐⭐

**행동 패턴**
- 화면 중앙에 자리잡고 투사체 난사
- 다단계 공격 패턴 보유
- 일정 체력 이하 시 분노 모드 진입
- 죽을 때 폭발로 작은 적 소환

**공격 패턴**
1. **원형 탄막**: 전방향으로 투사체 발사
2. **집중 공격**: 플레이어 방향 집중 사격
3. **광역 충격파**: 주기적인 넉백 공격
4. **소환**: 소형 적 2-4마리 소환

**단계별 변화**
- **Phase 1 (100%-60%)**: 기본 패턴
- **Phase 2 (60%-30%)**: 공격 빈도 증가
- **Phase 3 (30%-0%)**: 분노 모드, 모든 패턴 혼합

**스폰 특성**
- 단독 스폰
- 후반 웨이브 (15+)에서 등장
- 보스 웨이브의 서브 보스로 등장

**카운터 전략**
- 최대 화력 투입
- 장기전 준비 (체력 관리)
- 패턴 암기 후 안전 지대 활용
- 소환된 적은 무시하고 본체 집중

**디자인 포인트**
- "미니 보스"로 중간 난이도 체크포인트 역할
- 다양한 공격 패턴으로 플레이어의 종합 대응 능력 테스트
- 분노 모드로 긴장감 유지 및 극적 연출
- 보스 전의 예행연습 (패턴 학습)

## 5. 특수 적 - 엘리트 & 보스

기본 적 타입을 기반으로 한 강화 버전과 웨이브의 보스급 적들입니다.

### 5.1 엘리트 변종 (Elite Variants) - 모든 타입의 강화판

**공통 특성**
- 모든 기본 적 타입에 엘리트 버전 존재
- **외형**: 붉은색 또는 금색 + 빛나는 오라 효과
- **기본 능력 강화**:
  - 체력: 2-3배 증가
  - 공격력: 1.5-2배 증가
  - 이동 속도: 1.2-1.5배 증가
- **보상**: 경험치 2-3배, 드랍 확률 증가
- **등장 조건**:
  - 특정 웨이브 (5, 15, 25 등)에서 보장 등장
  - 웨이브 진행 중 랜덤 스폰 (확률 10-20%)
  - 보스 웨이브의 호위로 등장

#### 엘리트 소형 적

**Elite Dart (엘리트 다트)**
- **외형**: 붉은색 + 불꽃 트레일
- **기본 강화**: 체력 2배, 속도 1.3배, 공격력 1.5배
- **특수 능력**: 죽을 때 4방향으로 투사체 발사 (폭발 패턴)
- **위협도**: ⭐⭐☆☆☆ (기본 ⭐ → 엘리트 ⭐⭐)
- **전략적 의미**:
  - 단순한 돌진 적에서 죽음의 폭발을 가진 적으로 변화
  - 근접전이 위험해져 거리 유지 필요
  - 그룹으로 나올 경우 연쇄 폭발 주의

**Elite Splinter (엘리트 파편)**
- **외형**: 금색 + 전기 효과
- **기본 강화**: 체력 2.5배, 속도 1.4배, 공격력 1.5배
- **특수 능력**:
  - 분열 단계 +1 추가 (4단계까지)
  - 분열 시 생성 개수 증가 (2-3개 → 3-4개)
  - 최종 단계에서도 소형 폭발 발생
- **위협도**: ⭐⭐⭐☆☆ (기본 ⭐⭐ → 엘리트 ⭐⭐⭐)
- **전략적 의미**:
  - 분열 메커니즘의 극대화
  - 화면을 더욱 빠르게 채움
  - 관통 무기의 중요도 상승

#### 엘리트 중형 적

**Elite Asteroid (엘리트 소행성)**
- **외형**: 어두운 붉은색 + 균열에서 빛남
- **기본 강화**: 체력 3배, 공격력 2배, 회전 속도 증가
- **특수 능력**:
  - 파괴 시 더 많은 파편 생성 (3-5개 → 6-8개)
  - 주기적으로 작은 파편 방출 (살아있는 동안)
  - 파편들이 호밍 기능 보유
- **위협도**: ⭐⭐⭐☆☆ (기본 ⭐⭐ → 엘리트 ⭐⭐⭐)
- **전략적 의미**:
  - 단순 장애물에서 능동적 위협으로
  - 방치하면 화면을 파편으로 채움
  - 우선순위가 낮음에서 중간으로 상승

**Elite Bomber (엘리트 폭격기)**
- **외형**: 금색 장갑 + 에너지 충전 효과
- **기본 강화**: 체력 2.5배, 속도 1.3배, 공격력 2배
- **특수 능력**:
  - 투사체 발사 개수 증가 (2-3개 → 4-6개)
  - 투사체에 호밍 기능 추가
  - 발사 주기 감소 (더 자주 공격)
- **위협도**: ⭐⭐⭐⭐☆ (기본 ⭐⭐⭐ → 엘리트 ⭐⭐⭐⭐)
- **전략적 의미**:
  - 원거리 위협의 극대화
  - 회피만으로는 불충분, 적극적 제거 필요
  - 최우선 처치 대상으로 변경

**Elite Carrier (엘리트 운반선)**
- **외형**: 붉은색 선체 + 다수의 발광 포트
- **기본 강화**: 체력 3배, 속도 1.2배
- **특수 능력**:
  - 소환 주기 감소 (5-8초 → 3-5초)
  - 한 번에 더 많이 소환 (2-3마리 → 4-5마리)
  - Elite Dart 소환 가능 (확률 30%)
  - 최대 동시 소환 수 증가 (8-10마리 → 15마리)
- **위협도**: ⭐⭐⭐⭐⭐ (기본 ⭐⭐⭐⭐ → 엘리트 ⭐⭐⭐⭐⭐)
- **전략적 의미**:
  - 가장 위험한 엘리트 변종
  - 즉각 처치하지 않으면 게임 오버 수준
  - 소환 속도가 처치 속도를 초과
  - 보스급 우선순위

**Elite Dreadnought (엘리트 드레드노트)**

  - 꼬리 끝에서 작은 투사체 발사
- **위협도**: ⭐⭐⭐⭐⭐ (기본 ⭐⭐⭐⭐ → 엘리트 ⭐⭐⭐⭐⭐)

  - 완전 제거까지 시간 증가
  - 다중 위협 요소 (몸체 + 충격파 + 투사체)

**Elite Leviathan (엘리트 리바이어던)**
- **외형**: 보라색 + 차원 균열 효과

  - 주기적으로 화면 전체 왜곡 (플레이어 이동 방향 반전)
- **위협도**: ⭐⭐⭐⭐⭐ (기본 ⭐⭐⭐⭐⭐, 변화 없지만 극대화)
- **전략적 의미**:

  - 최대 화력과 최적 빌드 필수
  - 소환 적 무시 불가능 (엘리트 포함)
  - 왜곡 효과로 조작 난이도 극상

## 7. 우리 프로젝트 적용 방안

### 7.1 적 설계 원칙

#### 역할 다양화
- 최소 3-4가지 역할 구현
- 각 역할마다 명확한 특성
- 역할 조합으로 전략적 깊이 제공

#### 명확한 카운터
- 각 적에 대한 효과적인 무기/전략 존재
- 플레이어가 대응 방법을 학습할 수 있게
- "정답"이 아닌 "유리한 선택지" 제공

#### 시각적 차별화
- 크기, 색상, 모양으로 역할 구분
- 위험한 적은 더 눈에 띄게
- 공격 예고를 명확하게 표시

### 7.2 구현 우선순위

#### Phase 1: 기본 적 타입 (3종)
1. **Chaser** (Dart 기반)
   - 기본 추적 적
   - 직선 이동
   - 낮은 체력

2. **Shooter** (Bomber 기반)
   - 원거리 공격
   - 거리 유지
   - 투사체 발사

3. **Tank** (Asteroid 기반)
   - 높은 체력
   - 느린 이동
   - 장애물 역할

#### Phase 2: 특수 적 추가 (2-3종)
1. **Spawner** (Carrier 기반)
   - 소환 능력
   - 우선 처치 필요

2. **Splitter** (Splinter 기반)
   - 분열 메커니즘
   - 연쇄 반응

#### Phase 3: 고급 적 & 보스
1. 대형 적 (Dreadnought, Serpent 기반)
2. 보스 시스템 구현
3. 엘리트 변종 시스템

### 7.3 밸런싱 지침

#### 체력 밸런스
```
소형: 플레이어 DPS * 0.5-1초
중형: 플레이어 DPS * 2-4초
대형: 플레이어 DPS * 5-10초
보스: 플레이어 DPS * 30-60초
```

#### 공격력 밸런스
```
소형: 플레이어 HP의 5-10%
중형: 플레이어 HP의 10-20%
대형: 플레이어 HP의 20-30%
보스: 플레이어 HP의 30-50%
```

#### 스폰 빈도
- 소형: 5-10마리 / 웨이브
- 중형: 2-5마리 / 웨이브
- 대형: 0-2마리 / 웨이브
- 보스: 1마리 / 10웨이브
