# 벨런싱 체크리스트

> 플레이테스트 시 검증 항목

---

## 플레이어

### 생존력
- [ ] 첫 플레이 3분 이상 생존
- [ ] 실드 재생 속도 적절
- [ ] 피격 후 재생 지연 적절

### 공격력
- [ ] 기본 적 처치 3-5초
- [ ] 업그레이드 시 체감
- [ ] 풀 업그레이드 시 압도적

### 기동성
- [ ] 적 공격 회피 가능
- [ ] 조준 답답하지 않음

### 성장
- [ ] 30-60초마다 레벨업
- [ ] 업그레이드 선택 고민됨
- [ ] 시간에 따라 강해지는 느낌

---

## 적

### 체력 & 위협도
- [ ] 적별 체력 차이 체감
- [ ] 대형 적 확실히 강함
- [ ] 보스 압도적

### 공격력
- [ ] 초반 적 공격 치명적이지 않음
- [ ] 중반 적 위협적
- [ ] 후반 적 긴장감 제공
- [ ] 보스 공격 위험하지만 회피 가능

### 스폰
- [ ] 초반 밀도 적절
- [ ] 중반 긴장감 유지
- [ ] 후반 압박감
- [ ] 보스 등장 전 정리됨

---

## 난이도 곡선

### 초반 (0-3분)
- [ ] 조작법 익힐 시간 충분
- [ ] 1분 내 첫 레벨업
- [ ] 초보자도 3분 생존
- [ ] 지루하지 않음

### 중반 (3-7분)
- [ ] 적절한 도전 지속
- [ ] 업그레이드 효과 누적
- [ ] 선택에 따라 체감 다름
- [ ] 갑작스런 난이도 점프 없음

### 후반 (7-10분)
- [ ] 점점 어려워짐
- [ ] 방심하면 죽음
- [ ] 10분 도달 시 준비된 느낌
- [ ] 어렵지만 클리어 가능

### 보스전 (10분)
- [ ] 보스 등장 특별함
- [ ] 패턴 학습 가능
- [ ] 클리어 시 성취감

---

## 빌드 다양성

### 각 빌드 클리어 가능?
- [ ] 순수 사격 빌드
- [ ] 순수 충돌 빌드
- [ ] 순수 탱커 빌드
- [ ] 하이브리드 빌드

### 밸런스
- [ ] 특정 빌드가 압도적이지 않음
- [ ] 빌드별 약점 존재

---

## 플레이어 피드백

### 재미
- [ ] 몰입감 (시간 가는 줄 모름)
- [ ] "한 판 더" 하고 싶음
- [ ] 친구 추천 의향

### 공정성
- [ ] 실력으로 결과 차이
- [ ] 사망 시 "내 실수"
- [ ] 억울함 없음

### 도전성
- [ ] 어렵지만 깰 수 있을 것 같음
- [ ] 플레이할수록 실력 향상
- [ ] "다음엔 더 잘할 수 있어"

---

## 우선순위

**P0 (Critical)** - 즉시 수정
- 게임 진행 불가
- 심각한 밸런스 붕괴

**P1 (High)** - 빠른 수정
- 플레이어 불만 큼
- 난이도 곡선 문제

**P2 (Medium)** - 개선 필요
- 소소한 불균형
- UI/UX 개선

**P3 (Low)** - 여유 시
- 밸런스 미세 조정
- 최적화
