### 🔴 최우선 (P0)

#### 1. 10분 보스 클리어 시스템
- [ ] **10분 생존 시 최종 보스 출현 로직**
- [ ] **보스 격파 시 승리 조건 처리**
- [ ] **승리/패배 결과 화면**
- [ ] 보스 등장 연출 (화면 떨림, BGM 변화, 경고 메시지)
- **목적**: 명확한 클리어 목표 제공, 모바일 최적화된 플레이 타임

#### 2. 스폰 이벤트 경고 시스템
- [ ] 대량 스폰 전 화면 경고 표시 (텍스트/이펙트)
- [ ] 특정 방향 경고 UI (화살표/색상 표시)
- **목적**: 플레이어에게 대응 시간 제공, 긴장감 조성

#### 3. 게임 밸런스 조정
- [ ] **적 수치 재조정**
  - 적 체력/실드 밸런싱 (현재값 검토)
  - 적 피해량 조정 (플레이어 생존 시간 고려)
  - 적 점수(비용) 재평가
- [ ] **난이도 곡선 개선 (10분 기준)**
  - 초반(0-3분) 난이도 완화, 학습 구간
  - 중반(3-7분) 점진적 상승, 빌드 성장
  - 후반(7-10분) 피크 난이도, 보스전 준비
  - 10분: 최종 보스 출현 및 클리어
- [ ] **플레이 테스트 기반 조정**
  - 평균 생존 시간 측정
  - 레벨업 주기 검토
  - 업그레이드 밸런스 확인

#### 4. 오브젝트 풀링 시스템
- [ ] 발사체 풀링 (BulletBase)
- [ ] 적 풀링 (EnemyShip)
- [ ] FloatingText 풀링
- **목적**: 성능 최적화 (GC 부하 감소, 프레임 안정화)

### 🟠 높음 (P1)

#### 5. 플레이어 빌드 시스템 심화
- [ ] **업그레이드 옵션 확장** (현재 8개 → 15-20개)
  - 관통 (Penetration)
  - 멀티샷 (Multi Shot)
  - 발사체 속도/크기
  - 충돌 피해 증가
  - 실드 재생
  - 경험치 획득량 증가
  - 치명타 확률/피해
  - 넉백 강도
  - 드론 시스템
- [ ] **빌드 시너지 설계**
  - 사격 빌드 (High DPS): 멀티샷 + 연사 + 피해
  - 충돌 빌드 (Tank): 충돌 피해 + 체력 + 넉백
  - 기동 빌드 (Dodge): 이동속도 + 회전 + 재생
  - 하이브리드 빌드 조합
- [ ] **업그레이드 희귀도 시스템** (Common/Rare/Epic)

#### 6. UI/UX 개선
- [ ] 일시정지 메뉴

- [ ] 타이머 UI (10분 카운트다운)
