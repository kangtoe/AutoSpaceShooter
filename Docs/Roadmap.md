# Just 10 Minutes! - 개발 로드맵

> 최종 업데이트: 2026-01-24
> 게임 타이틀: **Just 10 Minutes!** (프로젝트명: AutoSpaceShooter)

## 📌 우선순위 표기

각 항목은 다음 우선순위로 구분됩니다:
- 🔴 **최우선**: 게임의 핵심 요소, 즉시 필요한 기능
- 🟠 **높음**: 중요하지만 약간의 유예 가능
- 🟡 **중간**: 개선 필요, 측정/검토 후 진행
- 🟢 **낮음**: 장기적 개선, 여유 있을 때 진행
- ✅ **완료**: 구현 완료된 시스템

## 🎯 현재 진행 상황

### ✅ 완료된 핵심 시스템
- 플레이어 조작 및 전투 시스템
- 시간 기반 스폰 시스템 (예산, 동적 난이도, 존재 점수)
- 특수 스폰 이벤트 시스템

- 14종 적 유형 & 다양한 발사체
- UI 시스템 (게임 UI, 업그레이드 선택, 게임 클리어 화면)
- 입력 시스템 (키보드/마우스/게임패드/모바일)

## 📋 개발 항목 (카테고리별)

### 🎮 게임 핵심 시스템

#### 플레이어 빌드 시스템 심화 🔴
- [ ] **업그레이드 옵션 확장** (현재 8개 → 15-20개)
  - 관통 (Penetration)
  - 멀티샷 (Multi Shot)
  - 발사체 속도/크기
  - 충돌 피해 증가
  - 실드 재생
  - 경험치 획득량 증가
  - 치명타 확률/피해
  - 넉백 강도
  - 드론 시스템
- [ ] **빌드 시너지 설계**
  - **사격 빌드**: 멀티샷 + 연사 속도 + 발사체 데미지 → 압도적 화력으로 원거리 제압
  - **충돌 빌드**: 충돌 피해 + 이동 속도 + 최대 실드 → 적에게 돌진하여 충돌로 격파
  - **탱커 빌드**: 최대 내구도/실드 + 실드 재생 + 충돌 피해 → 적과 부딪히며 근접 전투
  - **기동 빌드**: 이동/회전 속도 + 실드 재생 → 빠른 움직임으로 회피 중심
  - 하이브리드 빌드 조합
- [ ] **업그레이드 희귀도 시스템** (Common/Rare/Epic)
- **목적**: 재미와 리플레이성의 핵심, 다양한 플레이 스타일 제공

#### 전투 시스템 확장 🟡
- [ ] **적 AI 패턴 다양화**
  - 거리 유지형 (플레이어 회피)
  - 포위 공격형
  - 소환형 (드론 생성)
- [ ] **장애물 시스템** (선택적)

### ⚖️ 콘텐츠 & 밸런스

#### 게임 밸런스 조정 🔴
- [ ] **적 수치 재조정**
  - 적 체력/실드 밸런싱 (현재값 검토)
  - 적 피해량 조정 (플레이어 생존 시간 고려)
  - 적 점수(비용) 재평가
- [ ] **난이도 곡선 개선 (10분 기준)**
  - 초반(0-3분) 난이도 완화, 학습 구간
  - 중반(3-7분) 점진적 상승, 빌드 성장
  - 후반(7-10분) 피크 난이도, 보스전 준비
  - 10분: 최종 보스 출현 및 클리어
- [ ] **플레이 테스트 기반 조정**
  - 평균 생존 시간 측정
  - 레벨업 주기 검토
  - 업그레이드 밸런스 확인

#### 컨텐츠 추가 🟡
- [ ] **5분 미니보스 이벤트** 구현

- [ ] **리롤 시스템** (업그레이드 재생성)
