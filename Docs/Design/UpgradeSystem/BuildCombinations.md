# 빌드 조합 가이드

> Nova Drift 방식의 암묵적 시너지 - 업그레이드 조합을 통한 빌드 구축

## 개요

이 문서는 효과적인 **업그레이드 조합**을 제안합니다. Nova Drift처럼 명시적인 "시너지 시스템"은 없지만, 특정 업그레이드들을 함께 선택하면 **자연스럽게 강력한 조합**이 만들어집니다.

**설계 철학**:
- 게임이 "시너지 활성화!"라고 알려주지 않음
- 플레이어가 스스로 효과적인 조합을 **발견하는 재미**
- 모든 조합이 자연스럽게 작동 (강제 없음)

**참고**: [../../References/NovaDrift_BuildSystem.md](../../References/NovaDrift_BuildSystem.md)

---

## 빌드 아키타입

### 1. 탱커 빌드

**핵심 아이덴티티**: 적과 부딪히면서 싸우는 근접 전투

#### 효과적인 조합

**생존력 + 근접 공격**
- 최대 실드 + 최대 내구도 → 높은 총 체력
- 실드 재생 속도 + 실드 재생 지연 감소 → 지속적인 회복
- 충돌 피해 → 근접에서 추가 딜

**왜 효과적인가?**
- 높은 체력으로 적진에 진입 가능
- 빠른 실드 재생으로 연속 교전 가능
- 충돌 피해로 근접에서도 위협적

**플레이 스타일**:
적과 가까이 붙어서 싸우며, 실드가 소진되면 잠시 후퇴했다가 재생 후 재진입

#### 심화 조합

**탱커 + 기동**
- 위 조합 + 이동 속도
- 빠르게 접근하고 후퇴하는 "돌격 탱커"

**탱커 + 사격**
- 위 조합 + 멀티샷
- 근접 + 원거리 혼합형 "브루저"

---

### 2. 사격 빌드

**핵심 아이덴티티**: 원거리에서 압도적인 화력으로 제압

#### 효과적인 조합

**화력 집중**
- 멀티샷 + 연사 속도 → 단위 시간당 투사체 수 급증
- 발사체 피해 → 각 탄환의 위력 강화
- 발사체 속도 → 명중률 향상

**왜 효과적인가?**
- 멀티샷과 연사 속도는 **곱연산** 효과 (5발 × 2배 속도 = 10발/초)
- 발사체 속도로 빠른 적도 명중 가능
- 원거리에서 적을 녹임

**플레이 스타일**:
적과 거리를 유지하며 지속적으로 발사, 탄막으로 화면을 채움

#### 심화 조합

**사격 + 관통** (미구현)
- 위 조합 + 관통
- 밀집된 적 무리를 한 번에 처리

**사격 + 정확도**
- 멀티샷만 낮게 유지 + 발사체 속도/피해 집중
- 단일 타겟 저격형

---

### 3. 충돌 빌드

**핵심 아이덴티티**: 적에게 돌진하여 충돌로 격파

#### 효과적인 조합

**돌진 공격**
- 충돌 피해 + 이동 속도 → 빠른 돌진으로 충돌 피해 증가
- 최대 실드 + 실드 재생 → 충돌 시 받는 피해 버티기
- (선택) 회전 속도 → 기동성 향상

**왜 효과적인가?**
- 이동 속도가 높을수록 충돌 빈도 증가
- 실드로 충돌 피해 흡수
- 빠른 재생으로 연속 돌진 가능

**플레이 스타일**:
적을 향해 직진 돌진, 충돌 후 선회하여 다시 돌진 반복

#### 심화 조합

**충돌 + 넉백** (미구현)
- 위 조합 + 넉백 강도
- 적을 밀어내며 광역 제어

**충돌 + 질량** (미구현)
- 위 조합 + 질량
- 넉백 저항하며 충돌 피해 증가

---

### 4. 기동 빌드

**핵심 아이덴티티**: 빠른 움직임으로 적을 회피하며 견제

#### 효과적인 조합

**고속 회피**
- 이동 속도 + 회전 속도 → 민첩한 기동
- 실드 재생 속도 + 실드 재생 지연 감소 → 회피 중 재생
- (선택) 발사체 속도 → 빠른 이동 중에도 명중

**왜 효과적인가?**
- 높은 기동력으로 적의 공격 회피
- 피해를 받지 않으므로 실드가 빠르게 재생
- 계속 움직이며 공격 가능

**플레이 스타일**:
끊임없이 움직이며 적의 탄막을 회피, 틈틈이 공격

#### 심화 조합

**기동 + 사격**
- 위 조합 + 멀티샷/연사 속도
- "스피드 건너" 스타일

**기동 + 잔상** (미구현)
- 위 조합 + 애프터버너
- 잔상이 적을 공격

---

## 하이브리드 빌드

### 브루저 (탱커 + 사격)
**조합**: 최대 실드 + 실드 재생 + 멀티샷 + 발사체 피해

근접에서도 버티면서 화력도 강한 만능형. 어느 상황에서도 무난하게 대응 가능.

### 히트 앤 런 (충돌 + 기동)
**조합**: 충돌 피해 + 이동 속도 + 회전 속도 + 실드 재생

빠르게 돌진하여 충돌 후 빠르게 이탈. 연속 돌진으로 적을 제거.

### 기동 사수 (사격 + 기동)
**조합**: 멀티샷 + 연사 속도 + 이동 속도 + 회전 속도

빠르게 움직이며 탄막을 뿌림. 기동력으로 위험 회피.

### 균형형 (모든 스탯 골고루)
**조합**: 모든 카테고리에서 2-3레벨씩

뚜렷한 장점은 없지만 약점도 없는 안정적인 플레이.

## 빌드 구축 전략

### 초반 (Lv 1-5): 방향 설정
- 하나의 빌드 아이덴티티 선택 (탱커/사격/충돌/기동)
- 핵심 업그레이드 1-2개 먼저 올리기
  - 탱커 → 최대 실드
  - 사격 → 멀티샷 또는 연사 속도
  - 충돌 → 충돌 피해
  - 기동 → 이동 속도

### 중반 (Lv 6-15): 조합 구축
- 핵심 업그레이드와 **잘 어울리는 업그레이드** 추가
  - 탱커 → 실드 재생, 충돌 피해 추가
  - 사격 → 연사 속도, 발사체 피해 추가
  - 충돌 → 이동 속도, 실드 추가
  - 기동 → 회전 속도, 실드 재생 추가

### 후반 (Lv 16+): 빌드 완성
- 주력 업그레이드를 높은 레벨로 집중
- 약점 보완
  - 탱커의 낮은 화력 → 사격 업그레이드 추가
  - 사격의 낮은 생존력 → 실드 업그레이드 추가
  - 충돌의 위험성 → 실드 재생 추가
  - 기동의 낮은 화력 → 사격 업그레이드 추가

## 곱연산 효과 (Multiplicative Scaling)

Nova Drift처럼 특정 업그레이드 조합은 **곱연산**으로 효과가 증폭됩니다.

### 사격 화력
```
기본 DPS = 발사체 피해 × 멀티샷 × 연사 속도

예시:
- 발사체 피해: 20
- 멀티샷: 3발
- 연사 속도: 5발/초
→ DPS = 20 × 3 × 5 = 300

각각을 2배로 올리면:
→ DPS = 40 × 6 × 10 = 2400 (8배 증가!)
```

**교훈**: 한 가지만 집중하는 것보다 여러 요소를 골고루 올리면 곱연산 효과

### 생존력
```
유효 체력 = (실드 + 내구도) × (1 + 재생 효율)

재생 효율 = 재생 속도 / 재생 지연
```

빠른 재생은 단순히 HP를 올리는 것 이상의 효과를 냄

### 충돌 피해
```
충돌 효율 = 충돌 피해 × 충돌 빈도

충돌 빈도 = 이동 속도 × 기동성
```

이동 속도와 충돌 피해를 함께 올리면 곱연산 효과

## 업그레이드 선택 팁

### 좋은 선택
- ✅ 이미 선택한 업그레이드와 어울리는 것
- ✅ 빌드의 약점을 보완하는 것
- ✅ 곱연산 효과를 만드는 것

### 피해야 할 선택
- ❌ 빌드 방향과 무관한 업그레이드
- ❌ 한 가지만 극단적으로 올리기 (예: 멀티샷만 10레벨)
- ❌ 모든 업그레이드를 조금씩 찍기 (방향성 없음)

### 상황별 선택
**보스전이 가까울 때**:
- 생존력 업그레이드 우선 (최대 실드, 실드 재생)

**적이 밀집했을 때**:
- 광역 공격 업그레이드 (멀티샷, 관통)

**경험치가 부족할 때**:
- 경험치 보너스 업그레이드 (미구현)

- **방어형**: 내구도 증가, 화력 감소
